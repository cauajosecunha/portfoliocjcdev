# Portfólio Cauá José Cunha

Este é o repositório **portfoliocjcdev**, contendo o site de portfólio pessoal de Cauá José Cunha.

## Estrutura do Projeto

- **index.html** – Página inicial.
- **portfolio.html** – Página com projetos e trabalhos.
- **styles.css** – Estilos globais usados nas páginas.
- **images/** – Imagens do site (ex.: `eu.png`).

## Como Executar Localmente

1. Clone o repositório:

   ```bash
   git clone git@github.com:cauajosecunha/portfoliocjcdev.git
   cd portfoliocjcdev
   ```

2. Abra o arquivo `index.html` ou `portfolio.html` em um navegador web:

   ```bash
   start index.html   # Windows
   # ou simplesmente abra o arquivo via explorador de arquivos.
   ```

## Tecnologias Utilizadas

- HTML5
- CSS3 (sem frameworks externos)
- Imagens estáticas

## Contribuição

1. Fork o repositório.
2. Crie uma branch para sua feature ou correção:

   ```bash
   git checkout -b minha-feature
   ```

3. Commit suas mudanças e faça push:

   ```bash
   git add .
   git commit -m "Descrição da alteração"
   git push origin minha-feature
   ```

4. Abra um Pull Request.
